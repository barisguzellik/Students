## Derleme ve Çalıştırma (.NET CLI)
Build

UI Proje > Student.UI/Student.UI

```bash
cd Student.UI/Student.UI
dotnet build
```

Run

```bash
cd Student.UI/Student.UI
dotnet run
```

Default 
```bash
http://localhost:5000
```
